# 📸 Sportlight — Instagram Clone Built with React Native + Expo

Welcome to **Sportlight**, a modern Instagram clone built using **React Native** and **Expo**. The project mimics core social media features like photo sharing, feed browsing, user profiles, and more — all designed for mobile using Expo Go.

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/muhozajohn/sportlight.git
cd sportlight
```

### 2. Install dependencies

```bash
pnpm install
# or
npm install
# or
yarn install
```

### 3. Run the development server

```bash
npx expo start
```

From there, choose how to open the app:

* 📱 [Expo Go](https://expo.dev/go) on your Android/iOS device
* 🧪 [Android Emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
* 🧪 [iOS Simulator](https://docs.expo.dev/workflow/ios-simulator/)
* 💡 [Development Build](https://docs.expo.dev/develop/development-builds/introduction/)

---

## 📁 Project Structure

This project uses **file-based routing** with the [Expo Router](https://docs.expo.dev/router/introduction/). The main application code resides in the `/app` directory.

```
sportlight/
├── app/              # App screens and routes
├── assets/           # Static files like images and fonts
├── components/       # Reusable UI components
├── constants/        # Color palette, theme, etc.
├── hooks/            # Custom React hooks
├── utils/            # Helper functions and utilities
└── ...
```

---

## 🧼 Reset the Starter Template (Optional)

If you'd like to start from a fresh scaffold:

```bash
npm run reset-project
```

This will move the existing starter code to `app-example/` and initialize a new blank `app/` directory.
